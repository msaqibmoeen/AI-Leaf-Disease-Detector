# AI-Leaf-Disease-Detector
Built an AI Leaf Disease Detector web app for my AI course using a trained Ai/Python model (not hard-coded). It identifies plant type, disease, category, severity, and confidence from leaf images and generates a diagnosis (symptoms, causes, treatment, prevention) with secure login, upload, and results dashboard.
Pleased to share my AI Semester Project: 
AI Leaf Disease Detector 🌿
As part of my Artificial Intelligence course, I developed an AI-powered Leaf Disease Detection web application that automatically analyzes plant leaf images to identify diseases and provide meaningful insights.
This system is not hard-coded — it uses a trained Machine Learning / Deep Learning model that independently detects:
Plant type
Leaf disease
Disease category (e.g., fungal)
Severity level
Model confidence score
Along with predictions, the application generates a complete diagnosis summary, including:
🩺 Symptoms
⚠ Possible causes
💊 Treatment recommendations
🛡 Prevention guidelines
The project combines AI intelligence with a user-friendly web interface, featuring:
Secure Signup & Login system
Image upload & preview
Interactive results dashboard
🎯 Why this project matters:
 Early disease detection can significantly improve plant care and help reduce crop damage by enabling timely preventive actions.
This hands-on project strengthened my understanding of:
Applying AI to real-world problems
Computer Vision–based image classification
End-to-end AI system development (model → API → UI)
I’m open to feedback, suggestions, and collaboration opportunities to further improve the model accuracy and dataset quality.
